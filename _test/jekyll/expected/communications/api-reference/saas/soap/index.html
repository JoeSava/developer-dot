NpaNxx To PCode

This method returns the PCode for the specified NpaNxx value. If the NpaNxx value is invalid or has no EZtax jurisdiction the return will be NULL. Returns the PCode that corresponds to an NpaNxx value.

Api Endpoint
POST https://eztaxasp.billsoft.com/EZTaxWebService/EZTaxWebService.svc?NpaNxxToPCode

Headers
